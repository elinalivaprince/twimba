# Twitter Clone

A simple Twitter clone built as part of a Scrimba JavaScript course.

## Features
- Post tweets
- Like and retweet
- Reply to tweets
- Delete your own tweets
- Data saved with localStorage

## What I learned
- textarea
- .forEach()
- dataAttributes
- conditionally rendering styles
- NOT operator (!)
- CDNs
    - add icons
    - generate UUIDs

## Status
Learning project - still improving.
